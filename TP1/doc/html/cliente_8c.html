Detailed Description

Codigo fuente del cliente. Cada cliente se conecta a un servidor del que pide los datos de las estaciones.

Para la conexion se debe tener la IP y puerto del servidor. El comando utilizado es connect usuario:port Si el usuario ingresado es invalido la conexion muere. En caso de ser valido el usuario podra ejecutar los siguientes comandos: -listar -descargar nro estacion -diario_precipitacion no_estación -mensual_precipitacion no_estación -promedio variable -desconectar
